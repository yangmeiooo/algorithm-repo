1. 链表，其实就是 **断开链表和连接双向链表复杂一点**
2. 常用的技巧， 链表前面放一个空节点。  非常适合后面的操作。



### leetcode 206 反转链表

迭代解法：

```java
 public ListNode reverseList(ListNode head) {
      ListNode virhead = null;
      
      while(head != null) {
          ListNode nextNode = head.next; 
          head.next = virhead;
          virhead = head;
          head = nextNode;
      }
        return virhead;
    }
```

总结： 一个是空节点，二个是，按照正常人思维去解决这个问题。



递归解法：

```java
 ListNode staticNode = null;
    // 返回 给定一个链表头的链表，被反转后的头节点。
    public ListNode reverseList(ListNode head) {
      // 递归解法, 同一个函数被多次调用就完事了。
      // 😁😁
        if (staticNode == null) staticNode = head;
        ListNode virhead = null;
        if (head != null) {
            if (head.next != null){
                ListNode returnNode = reverseList(head.next);
                head.next.next = head;
                if (staticNode == head) head.next = null;
                return returnNode;
            } else {
                return head;
            }
        }
        return virhead;
    }
```

### leetcode 21 合并2个有序链表

```java
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        if (l1 == null) return l2;
        if (l2 == null) return l1;

        ListNode virhead = new ListNode();
        ListNode curNode = virhead;

        while(l1 != null && l2 != null) {
            if (l1.val <= l2.val) {
                curNode.next = l1;
                curNode = l1;
                l1 = l1.next;
            } else {
                curNode.next = l2;
                curNode = l2;
                l2 = l2.next;
            }
        }

        while (l1 != null) {
            curNode.next = l1;
            curNode = l1;
            l1 = l1.next;
        }

        while (l2 != null) {
            curNode.next = l2;
            curNode = l2;
            l2 = l2.next;
        }

        return virhead.next;
    }
```

总结： 弄清楚情况即可。
